# MFoto

Приложение для создания и управления фотокнигой.

## Описание

Сайт для фотографа в Новосибирске, специализирующегося на фотокнигах для семей, детских садов и школ. Позволяет пользователям создавать персонализированные фотокниги, загружая свои фотографии и выбирая из различных шаблонов и макетов.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Особенности

- Загрузка и организация фотографий
- Создание персонализированных фотокниг
- Интерактивный просмотр
- Экспорт в различные форматы
